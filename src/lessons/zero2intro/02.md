## 前言
欢迎收看第二章的内容，在这一章里我们会讲解如何调试配置Launchpad。首次开箱Launchpad，拿出后我们连接电脑，尽量避免使用拓展坞来连接。

## 安装USB驱动
第一，我们需要给Pad安装USB驱动，我们打开Novation的官网：[点此访问](https://novationmusic.com)，找到Support-Download，然后进入Launchpad的页面，选择你的设备，Novation USB Driver就是USB驱动

* Windows 10以上用户请安装上方最新的
* Windows 10以下用户请安装下方的旧版本

![img](/img/tutorial/zerostart/02/driver.png)

::: tip
如果你没法访问官网，这是由于中国国情所致导致无法访问网站的关键资源（涉及Google，Facebook）
请自行魔法上网解决或者使用本站为你提前准备的，可以前往下载：[跳转](/res/driver)
:::

下载好以后插上Pad，点开驱动一直下一步进行安装即可，完成后重启电脑

### 检查是否成功安装

打开搜索，输入“设备管理器”，找到“声音、视频和游戏控制器”，双击“Launchpad+型号”打开属性页面，检查驱动程序版本是否大于2.22，供应商是否是Novation
- 如果没问题你的驱动就正确安装了
  - 如果不是请点击下方的 “更新驱动程序”、“自动搜索驱动程序”，提示已安装后重启电脑即可

![img](/img/tutorial/zerostart/02/search.png)

![img](/img/tutorial/zerostart/02/devicemanager.png)

## 更新固件
第二，在游玩Launchpad前请确保你的固件是最新版本，如果是旧版本的固件可能会导致Bug，或者灯光不稳定之类的问题

::: warning
请务必在升级确保你已经安装好了USB驱动，并关闭
- 所有需要用到MIDI协议的软件（例如Ableton Live）
- 所有的杀毒软件
:::

### Launchpad MK3系列
MK3系列采用了WebApp的模式进行设备的管理，也就是Novation Components，只需要访问网页你就可以对Launchpad进行一系列的设置与修改，比方说添加自定义模式，更新固件等等，免去了采用软件的形式，当然它也有独立的软件版本，但它就是网页转应用（Electron），占用空间大，所以不如使用网页

#### 如何更新？
首先你需要访问[Novation Components](https://components.novationmusic.com/)，请注意使用Chrome或者Edge浏览器访问，其他浏览器如Safari无法访问MIDI设备
第一次打开请点击“Yes，I Agree”同意

![img](/img/tutorial/components-cookies.png =800x)

其次如果浏览器提示你访问MIDI设备请允许它，否则网页检测不到Launchpad

![img](/img/tutorial/chrome-allow-midi.png =400x)

然后插上你的设备，他会自动识别，点击进入设置页，这时如果你的固件版本过低，Components会自动弹出窗口提醒你更新，点击Update Firmware即可更新固件

![img](/img/tutorial/components-states.png =400x)

更新时需要你确认一下更新，他会提示你按哪个按键，按下它显示的按键即可开始升级，例如我的X是按Capture MIDI按钮，升级完成后Launchpad会自动重启

![img](/img/tutorial/components-update-agree.png =400x)

至此你就完成了升级

#### 我无法访问Novation Components，有别的方法吗？

Novation Components采用国外的服务器，国内由于国情访问速度非常慢甚至打不开，在这种情况下你可以尝试手动更新固件

::: warning
本文可能会有点超纲，请仔细观看理解
:::

首先你需要下载固件，[转到下载](/res/fwupdater#mk3手动更新固件)

下载完成后访问[Launchpad固件助手](https://lfu.arkitosekai.net)，等待网页加载好后敲键盘的箭头，输入：

`
↑，↑，↓，↓，←，→，←，→，
`

然后下拉框里就会出现“自定义SysEx文件”选项，原本是隐藏的

![img](/img/tutorial/lfu-sysex.png =200x)

选择你刚下载固件文件，确定

![img](/img/tutorial/lfu-choose.png =550x)

接着根据页面显示操作即可，之后你的Launchpad会自动重启

![img](/img/tutorial/lfu-bl.png =1000x)

至此你的固件升级完毕，如果卡在BL界面没有反应就拔了再插一遍，不要担心它刷成砖

### Launchpad MK2/MK1系列

MK2/MK1还维持着比较老的方法，每个固件都需要单独下载更新程序自己安装

你可以前往官网寻找你自己的型号后下载更新程序：[官网](https://downloads.novationmusic.com/novation)

如果你无法访问官网你可以直接在本站下载: [跳转](/res/fwupdater#launchpad-mk2系列)

下载好后插入Launchpad，然后打开升级程序，点击确定后等待他完成升级

![img](/img/tutorial/lppromk1-updater-progress.png =300x)

至此你就完成了升级

## 调整Launchpad设置
::: tip
此条仅Pro，X型号需要调整，其他无压力感应的型号请跳过
:::

### 关闭压力感应与触后
如果你是含有压力感应的型号，比方说Pro，X，你还需要关闭压力感应与触后。这里并不是关闭触发，压力感应可以使你敲击垫子的力度具现化，使其有轻重变化，而不会使你的力度被软件识别成只有按下跟松开。但开启压力感应后你在弹奏时的声音会忽大忽小，因为力度会影响音量，我们演奏灯光工程用不到，这是给演奏乐器音源用的，所以我们需要关闭

* Launchpad X，Pro MK3：

  首先进入Setup视图
  - Pro按住侧边的Setup按钮
  - X按住顶部Session按钮

  然后点击右边侧键第二个，切换到Velocity视图，然后点绿色的按钮，变红色后即可关闭压力感应

  ![img](/img/tutorial/zerostart/02/setup-vel.png)

  接着点击右边侧键第三个，切换到Aftertouch视图，然后点红色的按钮与它下方的按钮，即可关闭触后

  ![img](/img/tutorial/zerostart/02/setup-aft.png)

* Launchpad Pro MK1：

  进入Setup，点击第三排的第四个与第六个，然后点第四排的第6个，就可以关闭压感与触后

    ![img](/img/tutorial/zerostart/02/setup-vel-promk1.png)
