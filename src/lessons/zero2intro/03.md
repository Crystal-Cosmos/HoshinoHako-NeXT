## 前言
欢迎收看第三章的内容，在这一章里我们来讲Ableton Live相关的使用与调试方法。开始前需要说明的是，本文仅针对Launchpad用户，所以用不到的内容我都会一笔带过或者不讲，我只会讲Launchpad用户需要重点知道的板块

## Ableton Live使用教程
### 整体界面

![img](/img/tutorial/zerostart/03/gui-al12.png)

我们讲几个重点要先会使用的面板

### 菜单

![img](/img/tutorial/zerostart/03/menu-al12.png)

- 菜单栏：在这里会显示你当前工程的名字与工程文件的名字，下方的这些菜单你可以点开一个个查看，选项旁边写着对应的快捷键，我们主要会使用打开/保存工程，导出音频/MIDI
- BPM：每分钟节拍数，通常我们理解成曲速即可
- 节拍器：规律发出声音或视觉效果来显示节拍，通常用于录制演奏视频对节奏防止越弹越快
- 播放、停止、录制：三大金刚键，老三样，不多数
- 循环控制：控制在某个区域循环播放
- 随航：播放时的指针始终保持视图正中央，屏幕滚动播放
- MIDI映射：映射是一个很独特的概念，你们可以理解成自定义快捷键。演奏工程时翻页就是需要MIDI映射的，因为Launchpad本身没有翻页这个功能，翻页只是我们给它定义的快捷键，并不是Pad本身的功能。同理，其他没有定义但软件有的功能都可以通过映射达成
- CPU占用率指示条：顾名思义，CPU占用越高你的工程就越卡，同时伴随着爆音
- 旁边还有两盏灯，一个是CPU/硬盘过载灯，一个是是否检测到MIDI信号的指示灯

### 浏览器

![img](/img/tutorial/zerostart/03/browser-al12.png)

浏览器面板用于获取外部或者内部的素材，我们只需要了解4样东西；
- 乐器里的Drum Rack，Instrument Rack
- MIDI效果器里的MIDI Effect Rack
- Max for Live

这几样东西是构成Launchpad工程的全部

### Session/Arrangement视图

![img](/img/tutorial/zerostart/03/session-arrangement-al12.png)

这是Session与Arrangement视图，也是我们重点使用的视图

这两个视图大致是相同的，两者的区别只是针对的领域不同：
- Session常用于即兴或者现场创作Loop
- Arrangement则用于编排

你可以在右上角这里点击来切换页面，或者按Tab键，我们接下来所有的游玩与制作灯光工程都只使用Arrangement视图，所以我们切换它

![img](/img/tutorial/zerostart/03/arrangement.png)

在Arrangement视图中，你可以看到很多的一长横条，我们称之为轨道，可以添加与删除，每个轨道在默认下各自运行互不干扰，也可以相互调用

轨道有两种不同的分支
- 音频Audio：针对音频文件，像歌曲，采样，都是放在这个轨道上的
- MIDI：用于处理MIDI信号，比方说Launchpad按键数据输入与灯光数据输出

轨道还有一个控制区域
- 标题栏，你可以给每个轨道重命名用来区分，或者说右键给他选一个醒目的颜色，三角可以收起轨道
- IO面板：即输入与输出
  - 中间三个按钮是监听模式，只有打开了监听，你才能实时的检测输入与输出的变化
    - Auto：只有当开启预备录制时才会接收MIDI输入的信号，否则无反应（MIDI轨道中默认开启Auto模式）
    - In：实时监控，你的所有操作都会一直被接收并输出，但此时无法播放当前轨道内任何内容
    - Off：关闭监控（音频轨道中默认为Off）
- 调音台：
  - 轨道开关：黄色带数字的按钮，关闭后轨道不播放任何内容
  - Solo：S按钮，为代表独奏，开启后只有独奏的轨道能发出声音
  - 音量控制：蓝色的0的，控制的是当前轨道，按住上下拖动可以调整音量大小
  - 声道：C条，上下拖动可以调整左右声道位置
  - 预备录制：红色的小按钮，只有当你开启了这个按钮，你才能接收轨道为Auto监听的内容，同时启动单个轨道录音也需要开启，默认只能给一个轨道开启预备录制，你可以按住Ctrl实现多选。音频轨道中图标为实心圆，MIDI轨道中图标为圆中间带个音符图标

![img](/img/tutorial/zerostart/03/master.png)

Arrangement视图右下角还有一个轨道

- Master：通常称其为总线，所有的声音最后都会整合发送到这个轨道，然后再发送给声卡，我们才能听到声音
  - 总音量控制：淡蓝的0，控制整个工程的音量
  - “预览”音量控制：深蓝色的0，不过控制的是浏览器面板的预览音量

![img](/img/tutorial/zerostart/03/volume-midi.png)

所有轨道的最右边还有竖条或者圆圈
- 圆圈为MIDI轨道：代表MIDI信号是否接收
- 竖条为音频轨道：代表音频的音量（电平）

![img](/img/tutorial/zerostart/03/arrange-guitoggle.png)

界面右下角的圆形橙色按钮则是上文这些面板的显示与否，如果你发现你的面板不见了可从这里打开

### 机架

![img](/img/tutorial/zerostart/03/rack.png)

下方一长条称之为机架，等同于内容库，每个轨道都有一个机架，你需要在浏览器寻找效果器，然后拖进去，就可以赋予轨道内容

以下是一些机架的使用说明：
- 音频轨道机架只能放音频相关的东西，MIDI轨道机架只能放MIDI相关的东西，放错了最底下的状态栏会提示
- 机架有一个信号传递的逻辑，其信号是从最前往最后一层一层向后传递的，是串联与叠加的，并不是并联或者单独的，前面的信号如果改变了则会影响到后面直至输出。所以插件与效果器摆放的位置是有讲究的

### 状态栏

![img](/img/tutorial/zerostart/03/status.png)

软件的下方有一长条状态栏，用来显示当前工程的属性，或者其他什么东西遇到错误这里会以橙色高亮显示，提示你工程有问题，文件丢失，或者操作有误

### 其他

![img](/img/tutorial/zerostart/03/gui-2-al12.png)

每个面板的角落都有一个三角，这些是面板的收起按钮，点击以后则可以把无用的面板暂时收纳起来

### 帮助
如果你有上述没提到的面板不懂，可以将鼠标指针指到对应的面板或者功能键上，最下角的帮助面板会显示详细的功能介绍。如果没有显示这个面板，可以点击整个软件左下角的箭头按钮，即可调出

![img](/img/tutorial/zerostart/03/helpinfo-al12.png)

## 调整Live的偏好设定
偏好设置一共有9个设置项：
- 界面
- 主题与颜色
- 音频
- MIDI设备
- 文件与缓存
- 用户库目录
- VST插件
- 录音
- 许可证与维护项

### 界面
![img](/img/tutorial/zerostart/03/preference1-al12.png)

在界面选项卡里，你可以
- 设置Live的语言，Live版本大于等于10.1.13支持简体中文
- 可以重置所有的警告窗口
- 开启多开模式
- 可以设置Live的主题

依据喜好调整

### 主题与颜色
![img](/img/tutorial/zerostart/03/preference2-al12.png)

在主题与颜色选项卡里，你可以
- 调整黑暗模式或明亮模式
- 调整UI颜色，色调，对比度
- 开关轨道的颜色分配

### 音频
![img](/img/tutorial/zerostart/03/preference3-2-al12.png)

在音频选项卡里，请确保
- 音频的输出设备是你电脑当前正在使用的设备，你可以右键电脑右下角的喇叭按钮，打开声音设置，查看你当前选中的音频设备并在Live里设置相同的设备 
  :::tip 提醒
  注意需要选择后缀为DX的音频设备而非Wave，否则影响输出效果
  :::

  ![img](/img/tutorial/zerostart/03/audiooutput-al12.png)

- 将音频输出缓冲区设置为1024，默认为4096，这一步的作用是降低音频输出到耳机或者音响里的延迟，如果延迟过大将会极大的影响你演奏时的流畅度，因为当你按下按钮到音频输出时中间有40ms左右的延迟。如果设置后出现兹拉兹拉的声音或者爆音，请更换电脑
 
  ![img](/img/tutorial/zerostart/03/buffersize-al12.png)

- 如果你拥有声卡并且支持ASIO

  ![img](/img/tutorial/zerostart/03/preference3-asio-al12.png)

可以在上方的驱动程序类型里改成ASIO并选择你的声卡，并在声卡控制面板里设置Buffer Size，则可以以极低的延迟输出声音

此处的音频影响你后期弹奏时的手感，建议能在范围内拉到多低就拉到多低，否则容易手癖，个人平常都是声卡输出然后64 Samples缓冲区，基本延迟就2ms，如果没有声卡这个是做不到的

### MIDI设备
![img](/img/tutorial/zerostart/03/preference4-al12.png)

在MIDI设备选项卡里，我们需要一些设置让Live正确的识别我们的Pad，正常情况下你第一次接上设备开启Live时它会自己设置，但是有的时候它并不会，那我们就需要手动设置

- 我们需要在控制界面列里选择Launchpad，这里需要看你的型号进行选择，X则选择Launchpad X，Pro MK3则选择Launchpad Pro MK3，MK2（RGB）选择Launchpad MK2，以此类推，注意不要选择成Launchkey了

  然后将后方输入跟输出列这样设置（仅列出有彩色LED型号的设备）
  - Launchpad Pro MK3：
    - Windows：MIDIIN3/MIDIOUT3 （LPProMK3 MIDI）
    - macOS：Launchpad Pro MK3 （LPProMK3 DAW）
  - Launchpad X：
    - Windows：LPX MIDI
    - macOS：Launchpad X （LPX DAW Out）
  - Launchpad Mini MK3：
    - Windows：LPMiniMK3 MIDI
    - macOS：Launchpad Mini MK3 (LPProMK3 DAW Out)
  - Launchpad Pro MK1：
    - Windows：Launchpad Pro
    - macOS：Launchpad Pro（Live Port）
  - Launchpad MK2：
    - Windows：Launchpad MK2
    - macOS：Launchpad MK2
  - 想知道为什么这么设置请跳转：[此处](/lessons/others/midiportset)
- 上方控制面板设置完成后，还需要设置下方MIDI Ports，启用对应端口的轨道Track与外部控制Remote，其余的不开启，如下
  - Launchpad Pro MK3：
    - Windows：LPProMK3 MIDI
    - macOS：Launchpad Pro MK3 （LPProMK3 MIDI）
  - Launchpad X：
    - Windows：MIDIIN2/MIDIOUT2 （LPX MIDI）
    - macOS：Launchpad X （LPX MIDI Out）
  - Launchpad Mini MK3：
    - Windows：MIDIIN2/MIDIOUT2 （LPMiniMK3 MIDI）
    - macOS：Launchpad Mini MK3 (LPProMK3 MIDI Out)
  - Launchpad Pro MK1：
    - Windows：Launchpad Pro
    - macOS：Launchpad Pro（Live Port）
  - Launchpad MK2：
    - Windows：Launchpad MK2
    - macOS：Launchpad MK2
  - 示例（Launchpad Pro MK1 with CFW）：

  ![img](/img/tutorial/zerostart/03/midiports-al12.png)
- 如有Microsoft GS Wavetable Synth请无视，不要开启不要动

完成后你的Launchpad就设置好了，如果你的端口名字为红色，则可能你没有正确安装驱动或者表示你的设备被其他需要用MIDI的设备占用了，Live无法调用，请检查你的USB驱动是否安装，以及关闭其他正在同时使用MIDI设备的软件，或者重新拔插Launchpad

### 文件与缓存
![img](/img/tutorial/zerostart/03/preference5-al12.png)

在文件与缓存选项卡里，你可以
- 修改Live的缓存保存位置，当你放入音频文件时，部分文件会生成一个缓存文件扔在这个文件夹里
- 更改Max为外部安装的独立版
- 开关分析文件（每第一次加载音频时旁边跟着的`.asd`文件）

依据喜好调整

### 用户库
![img](/img/tutorial/zerostart/03/preference6-al12.png)

在用户库选项卡里，不建议修改任何设置

但如果你的用户库路径是空的，请设置成%USERPROFILE%\Documents\Ableton\User Library （如图）

### VST插件
![img](/img/tutorial/zerostart/03/preference7-al12.png)

在VST插件选项卡里，你可以设置VST2以及VST3插件的路径

### 录音
![img](/img/tutorial/zerostart/03/preference8-al12.png)

在录音选项卡里，推荐把“在Clip边缘创建淡化”这个选项关掉，否则音频的头跟尾会被淡入淡出，影响音频

### 许可证与维护项
![img](/img/tutorial/zerostart/03/preference8.png)

在许可证与维护选项卡里，可以关闭“自动更新”

以上就是大致需要设置的了
