## 简介
通过本收费一对一能够让你学会基础的工程制作方法，包括但不限于以下内容：
- Live的使用教程与基本设置
  - 连接Launchpad
  - 延迟调整
  - 优化
- 怎么找歌，怎么识别歌曲BPM
- 基本的工程框架
- 制作音频轨道
  - 音频轨道的布局
  - 提取分轨（基于UVR）
  - 切音
  - 音频修正
  - 翻页映射
- 制作灯光轨道
  - 灯光轨道的布局
  - 基本灯光制作（基于Iris/Resonator）
  - 复杂灯光绘制（基于Eyedrop）
- 进阶内容：
  - 一键多音/一键多灯
  - 灯光分层
  - 使工程兼容不同型号设备
- 整理与分发

不包含以下内容：
- Session工程制作
- 基于Live内建MIDI效果器的灯光绘制

免费提供所有常用软件与插件
