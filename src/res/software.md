## Ableton Live

Ableton Live是由Ableton公司制作的一款数字音频工作站（DAW），通过Live可以自由的创作音乐，现场演出，即兴演奏。Live与Launchpad做了深度整合

![img =180x](/images/software/live_suite.ico)

Live目前有以下版本：
- Intro 入门版
- Standard 标准版
- Suite 完整版
- Lite 精简版（与设备一起捆绑，不单独售卖）
- Trial 试用版（30天试用版的Suite，功能相同）

::: tip
- 硬件设备捆绑的Lite激活到Ableton账户后可以享受85折升级到Suite
- 仅有Suite版本带Max For Live模块，游玩Launchpad工程请务必购买Suite版本
- 试用版可90天免费试用完整内容，到期后可继续试用但无法保存/另存为，所以不要拿来做工程（
- 只有Live 10.1.13以上的版本兼容Launchpad MK3全系且含简体中文·
- 请不要冲动消费，先用试用版体验也是没有问题的_(:з」∠)_
:::

::: details Live的最低配置要求(Live 12版本)
::: tabs 
@tab Windows

* Windows 10（1909及更高版本）
* Intel® Core™ i5 处理器或 AMD 多核处理器。
* 8 GB 内存
* 1366x768 显示分辨率
* 支持 Link 的 ASIO 兼容音频硬件(同时可保证高品质的音频性能)
* 连接网络，获得 Live 授权 (建议使用高速网络连接，以便更快速地下载额外文件或更新 Live)
* 在进行基本安装时，系统磁盘上需要大约 3 GB 的空间 (建议预留 8 GB 的可用磁盘空间)
* 需要最多 76 GB 的磁盘空间来储存额外的音色文件 _(可选)_

@tab macOS

* macOS High Sierra 10.13 至 Sonoma 14
* Intel® Core™ i5 处理器或 Apple Silicon
* 8 GB 内存
* 1280x800 显示分辨率
* 建议使用与 Core Audio 兼容的音频接口
* 连接网络，获得 Live 授权 (建议使用高速网络连接，以便更快速地下载额外文件或更新 Live)
* 在进行基本安装时，系统磁盘上需要大约 3 GB 的空间 (建议预留 8 GB 的可用磁盘空间)
* 需要最多 76 GB 的磁盘空间来储存额外的音色文件 _(可选)_

:::
